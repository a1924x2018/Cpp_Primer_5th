## 随笔

- 类的成员函数或非成员函数是类接口的组成部分，则定义在同一头文件。
- 定义使用的成员必须在之前定义
  例如，我们必须按照如下方式设计程序：
  (1)首先定义Window_mgr类，其中声明clear函数，但是不能定义它。在clear使用Screen的成员之前必须先声明Screen。
  (2)接下来定义Screen，包括对于clear的友元声明。
  (3)最后定义clear，此时它才可以使用Screen的成员。
